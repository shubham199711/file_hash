# A Simple python code to create file hash and check if both are equal
----------
## package used
- hashlib
- os
- argparse

To create file hash
```bash
python3 createFileHash.py -f ./example.txt
```

To check file hash
```bash
python3 createFileHash.py -f ./example.txt -c 17d3dd954013cd207d77e9607ee628692eb9e634435276d8141d1235d0f97306
```
